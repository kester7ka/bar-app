# Bar Manager

Минималистичное веб-приложение для бара: учёт позиций, контроль сроков
годности и просмотр графика смен.

## Что умеет

- **Позиции** — добавление по штрихкоду (камера) или вручную; категории
  (ингредиенты, сиропы, печенье, прочее); открытие и закрытие позиций
  с учётом времени; цветовая подсветка истекающего и просроченного.
- **Главная** — что портится сегодня и завтра, часы, погода,
  быстрая сводка по бару.
- **График смен** — кто сегодня работает, расчёт примерной выплаты
  (до вычета налогов).
- **Инструменты** — калькулятор сроков годности (до минут), быстрый
  поиск по TOB, экспорт в CSV, статус сервисов.
- **Профиль** — QR-код для прихода на работу, тёмная/светлая тема,
  работа в офлайне (помнит вход, даже если сервер не отвечает).

## Как попасть в приложение

Вход — только по **одноразовому 8-значному ключу**. Ключ выдаёт админ
бара. После регистрации заходишь обычным логином и паролем.

## Установка на телефон

Открой `https://kester7ka.github.io/bar-app/` в браузере телефона и
добавь на главный экран — будет работать почти как нативное приложение,
с камерой и офлайн-режимом.
